# Nasa Rover App
An application that displays photos taken by NASA's rover vehicles on the surface of Mars.

## Tech Stack 
- MVVM
- Data Binding , View Binding
- Live Data
- Retrofit
- RxJava
- Dagger Hilt
- Glide

## API
API Link : [Api Link](https://api.nasa.gov/)
